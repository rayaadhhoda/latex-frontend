# Quick start

In this guide, we'll get the application set up and get started with authoring our first document.

## Installation

/// tab | MacOS

1. Download and open the installer.
2. Drag the icon to the `Applications/` folder.
3. You can now find it within your programs or via Spotlight search.

///

/// tab | Windows

1. Download and open the installer.
2. Open the application.

///

/// tab | Linux-based OS

1. Download the AppImage file.
2. Move it to your desired location and open it.
